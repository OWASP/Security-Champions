# Contributing to OWASP Security Champions Guide

The Security Champions Guide is an open-source community project,
and we are always delighted to welcome new contributors!

This repository is for the [Security Champions Guide](https://securitychampions.owasp.org/) site.

When contributing:

* see if there is [already an issue](https://github.com/OWASP/Security-Champions/issues) for what you want to do
* follow our [Code of Conduct](code_of_conduct.md)

## Content requests

If you have a suggestion for new content or changes to existing content
then [raise an issue](https://github.com/OWASP/Security-Champions/issues/new/choose) to request this.

### Project pages

If you instead want to change the content of the [OWASP project pages](https://owasp.org/www-project-security-champions-guidebook/)
then navigate to the [source repository](https://github.com/OWASP/www-project-security-champions-guidebook/)
to view and modify that content.
